# リアルタイム物体検出アプリ

FlaskとYOLOv8nを活用したリアルタイム物体検出アプリケーション。ウェブカメラからの映像に対してリアルタイムで物体検出を行い、検出した物体についての説明を表示します。

## 主な機能

- ウェブカメラからのリアルタイム映像表示
- YOLOv8nを使用した高精度な物体検出
- 検出した物体に対する説明文の自動生成
- シンプルで直感的なユーザーインターフェース

## 必要な環境

- Python 3.8 以上
- ウェブカメラ（内蔵カメラまたはUSBカメラ）
- 仮想環境（推奨）

## セットアップ

1. リポジトリをクローンまたはダウンロード

2. 仮想環境を作成して有効化：
```bash
python3 -m venv .venv
source .venv/bin/activate  # Linux/Macの場合
# または
.venv\Scripts\activate  # Windowsの場合
```

3. 必要なライブラリをインストール：
```bash
pip install -r requirements.txt
```

## 実行方法

仮想環境を有効化した状態で、以下のコマンドを実行：

```bash
# Python 3の場合
python3 app.py

# または
python app.py
```

アプリケーションが起動したら、ブラウザで `http://127.0.0.1:5000` にアクセスしてください。

## 使い方

1. 画面上部の「カメラを開始」ボタンをクリックしてカメラを起動します。
2. カメラが起動すると、リアルタイムで物体検出が始まります。
3. 検出された物体のバウンディングボックスをクリックすると、その物体についての説明が表示されます。
4. 「カメラを停止」ボタンでカメラを停止できます。

## APIエンドポイント

- `GET /`: メインのウェブインターフェースを表示
- `POST /object-detection`: 画像データを受け取り、物体検出結果をJSONで返す
- `POST /description`: 検出された物体の説明を生成して返す

## プロジェクト構造

```
pbl_ObjectDetection/
├── app.py                 # Flaskアプリケーションのメインファイル
├── requirements.txt       # プロジェクトの依存関係
├── .gitignore            # Gitで無視するファイル・ディレクトリ
├── config.py             # アプリケーション設定
├── detection.py          # YOLOv8nを使用した物体検出ロジック
├── description.py        # 物体説明生成ロジック
├── static/               # 静的ファイル（CSS, JavaScript, 画像など）
└── templates/            # HTMLテンプレート
    └── index.html        # メインのウェブインターフェース
```
